Elite Setup

If you want the best Windows experience, do this:

  • ‚úÖ Disable Startup Junk (Task Manager > Startup)

    Navigate to Task Manager by pressing Ctrl + Shift + Esc, go to the Startup tab, and disable any programs that are not necessary to launch at startup. This reduces boot time and background processes

  • ‚úÖ Turn Off Background Apps (Settings > Privacy > Background Apps)

    Navigate to Settings > Privacy > Background Apps. Toggle off the switch for "Let apps run in the background." This prevents unnecessary apps from using system resources when you're not actively using them, reducing memory usage and improving overall performance

  • ‚úÖ Use O&O ShutUp10++ (Disables bloatware & telemetry)

    Download and run O&O ShutUp10++ (a free privacy tool). It provides a comprehensive list of privacy settings, including the disabling of unnecessary telemetry, background services, and bloatware apps that send data back to Microsoft. Use this tool to fine-tune your system‚Äôs privacy by disabling services that aren‚Äôt essential for the user experience

  • ‚úÖ Enable Ultimate Performance Mode (

    powercfg -duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61
    )

    Open Command Prompt as Administrator. Enter the command: 

    powercfg -duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61
    . This activates the Ultimate Performance Mode, a setting designed for high-end systems to ensure maximum performance by reducing power management throttling. It‚Äôs not available by default on most consumer systems, so this command enables it

  • ‚úÖ Set Pagefile Manually (Virtual Memory)

    Go to Control Panel > System > Advanced System Settings > Settings (under Performance) > Advanced > Virtual Memory. Uncheck Automatically manage paging file size for all drives and manually set the initial and maximum sizes for the pagefile based on your system's RAM. A good rule of thumb is setting the pagefile size to 1.5x to 3x of your system's RAM to avoid excessive disk usage during high memory tasks

  • ‚úÖ Defrag HDD / Trim SSD (Defragment & Optimize Drives)

    Open Defragment and Optimize Drives from the Start menu. For HDDs, select the disk and click Optimize to defragment it. For SSDs, click Optimize to run the Trim operation, which helps the SSD clean unused blocks and maintain optimal performance. SSDs don't need traditional defragmentation but should be trimmed periodically to maintain speed

  • ‚úÖ Use WizTree to Find Large Files (Clear disk space)

    Download and run WizTree to scan your drives for large files and folders. It visually maps the files on your disk to show you which files are taking up the most space. Use it to identify unnecessary files that can be deleted or moved to free up valuable disk space

  • ‚úÖ Optimize Gaming Settings (Disable Game DVR, enable GPU scheduling)

    Open Settings > Gaming > Captures and turn off Record in the background while I'm playing a game (Game DVR). This prevents the Game DVR feature from using system resources and background processing while gaming. Enable Hardware-Accelerated GPU Scheduling (Settings > System > Display > Graphics Settings) to offload GPU scheduling to the dedicated GPU, improving performance in games

  • ‚úÖ Disable Search Indexing & SysMain (Superfetch)

    Open Services (type services.msc in Start Menu). Find and disable Windows Search (Search Indexing). This can improve performance by stopping unnecessary indexing in the background. Disable SysMain (Superfetch), which can sometimes consume resources unnecessarily when trying to optimize memory usage

  • ‚úÖ Run SFC, DISM, and CHKDSK to Fix Corruption

    Run SFC (System File Checker) to check and repair corrupted system files: Open Command Prompt as Administrator and type: 

    sfc /scannow
    . Use DISM (Deployment Imaging Service and Management Tool) to fix Windows corruption: Type: 
    DISM /Online /Cleanup-Image /RestoreHealth
    in Command Prompt. Run CHKDSK to check your disk for errors: Type: 
    chkdsk /f
    and press Enter. Restart the system if it prompts

  • ‚úÖ Use Process Lasso to Control CPU Usage

    Install Process Lasso to give you more control over CPU usage. It helps prioritize and limit processes that are consuming too many resources, preventing bottlenecks and ensuring that important applications or games get priority processing time. Use the tool to set CPU affinities and limits, as well as auto-balancing tasks for optimized performance

  • ‚úÖ Use TCP Optimizer for Better Internet Speeds

    Download TCP Optimizer and use it to optimize your internet settings. This tool helps tweak network settings such as MTU, RWIN (TCP Receive Window), and DNS settings, improving overall internet speed and reducing latency. You can select a pre-configured optimal setting or customize the settings based on your connection type and hardware
