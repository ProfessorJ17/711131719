Tutorial: Bitcoin Private Key Checker with Simulated Quantum Random Number Generation

This version replaces the IBM QRNG API with a simulated quantum random number generator while maintaining the same core functionality for checking Bitcoin private keys. This approach removes the dependency on external APIs and allows for offline operation while still simulating the benefits of quantum randomness for key generation.

Simulated Quantum Random Number Generator

Our simulated quantum random number generator is designed to mimic some aspects of quantum randomness using standard cryptographic methods. It aims to provide a more unpredictable source of randomness compared to typical pseudo-random number generators. Here are the key features:

  • Use crypto.randomBytes as a base entropy source: We leverage the built-in crypto.randomBytes module in Node.js to obtain cryptographically secure random bytes. This serves as the foundation for our simulated quantum randomness, ensuring a high level of initial entropy.
  • Apply quantum-like distribution patterns: The generator applies mathematical operations inspired by quantum mechanics to the base entropy. This includes bitwise operations and state manipulations designed to distribute the randomness in a way that resembles quantum distributions.
  • Implement superposition-like effects: To simulate quantum superposition, the generator uses a stateful approach where previous outputs influence future outputs in a complex manner. This is achieved through XOR operations and state shifting, creating an effect similar to quantum interference.
  • Ensure uniform distribution across the range: Despite the quantum-like simulations, the generator is engineered to ensure that the final output is uniformly distributed across the desired range. This is crucial for applications like private key generation where bias in randomness could lead to vulnerabilities.

Complete Code

Below is the complete JavaScript code for the Bitcoin Private Key Checker with the Simulated Quantum Random Number Generator. You can copy and paste this code into a file named script.js and run it using Node.js.

const puppeteer = require('puppeteer');
const fs = require('fs');
const path = require('path');
const crypto = require('crypto');

// Configuration
const puppeteer = require('puppeteer');
const fs = require('fs');
const path = require('path');
const crypto = require('crypto');
const csvWriter = require('csv-writer').createObjectCsvWriter;

// Configuration
const CONFIG = {
    BASE_URL: 'https://privatekeyfinder.io/private-keys/bitcoin/',
    OUTPUT_FILE_TXT: path.join(__dirname, 'btc.txt'),
    OUTPUT_FILE_JSON: path.join(__dirname, 'btc.json'),
    OUTPUT_FILE_CSV: path.join(__dirname, 'btc.csv'),
    MAX_PAGE_NUMBER: BigInt('1929868153955269923726183083478131797547292737984581739710086052358636024906'),
    PAGE_LOAD_TIMEOUT: 60000,
    MIN_WAIT_TIME: 4000,
    MAX_WAIT_TIME: 8000,
    BREAK_INTERVAL: 3 * 60 * 1000,
    BREAK_DURATION: 20000,
    MAX_RETRIES: 3,
    SIMULATED_QUANTUM_BITS: 256
};

// CSV Writer setup
const csvWriterInstance = csvWriter({
    path: CONFIG.OUTPUT_FILE_CSV,
    header: [
        {id: 'timestamp', title: 'Timestamp'},
        {id: 'address', title: 'Address'},
        {id: 'balance', title: 'Balance'},
        {id: 'privateKey', title: 'Private Key'}
    ],
    append: true
});

class SimulatedQuantumRNG {
    constructor() {
        this.seed = BigInt('0x' + crypto.randomBytes(32).toString('hex'));
        this.state = this.seed;
    }

    simulateSuperposition(bits) {
        const buffer = crypto.randomBytes(bits / 8);
        let result = BigInt(0);

        for (let i = 0; i < buffer.length; i++) {
            const byte = buffer[i];
            this.state = (this.state << BigInt(1)) ^ BigInt(byte);
            result = (result << BigInt(8)) | BigInt(byte);
        }
        return result;
    }

    generateQuantumRandom(maxNumber) {
        const rawRandom = this.simulateSuperposition(CONFIG.SIMULATED_QUANTUM_BITS);
        const normalized = rawRandom % maxNumber;
        const noise = BigInt('0x' + crypto.randomBytes(16).toString('hex'));
        const finalNumber = (normalized + noise) % maxNumber;
        return finalNumber.toString();
    }
}

class BitcoinKeyChecker {
    constructor() {
        this.checkedPages = new Set();
        this.browser = null;
        this.page = null;
        this.startTime = Date.now();
        this.qrng = new SimulatedQuantumRNG();
        this.jsonData = this.loadExistingJsonData();
    }

    loadExistingJsonData() {
        try {
            if (fs.existsSync(CONFIG.OUTPUT_FILE_JSON)) {
                const data = fs.readFileSync(CONFIG.OUTPUT_FILE_JSON, 'utf8');
                return JSON.parse(data);
            }
            return [];
        } catch (error) {
            console.error(`Error loading JSON data: ${error.message}`);
            return [];
        }
    }

    validateData(address, balance, privateKey) {
        return {
            isValid: typeof address === 'string' &&
                    typeof balance === 'number' && balance > 0 &&
                    typeof privateKey === 'string' &&
                    address.length > 0 &&
                    privateKey.length > 0,
            errors: []
        };
    }

    async saveData(address, balance, privateKey) {
        const timestamp = new Date().toISOString();
        const entry = { timestamp, address, balance, privateKey };

        const validation = this.validateData(address, balance, privateKey);
        if (!validation.isValid) {
            console.error(`Invalid data: ${JSON.stringify(entry)}`);
            return false;
        }

        try {
            // Save to TXT
            const txtEntry = `${timestamp} | ${address} | BAL: ${balance} | Private Key: ${privateKey}\n`;
            fs.appendFileSync(CONFIG.OUTPUT_FILE_TXT, txtEntry);

            // Save to JSON
            this.jsonData.push(entry);
            fs.writeFileSync(CONFIG.OUTPUT_FILE_JSON, JSON.stringify(this.jsonData, null, 2));

            // Save to CSV
            await csvWriterInstance.writeRecords([entry]);

            console.log(`Saved: ${address} | BAL: ${balance} | Private Key: ${privateKey}`);
            return true;
        } catch (error) {
            console.error(`Error saving data: ${error.message}`);
            return false;
        }
    }

    async initialize() {
        try {
            this.browser = await puppeteer.launch({
                headless: false,
                devtools: true,
                args: ['--start-maximized']
            });
            this.page = await this.browser.newPage();
            console.log('Browser initialized successfully');
        } catch (error) {
            throw new Error(`Failed to initialize browser: ${error.message}`);
        }
    }

    getQuantumRandomPageNumber() {
        try {
            return this.qrng.generateQuantumRandom(CONFIG.MAX_PAGE_NUMBER);
        } catch (error) {
            console.warn(`QRNG simulation failed: ${error.message}. Using fallback...`);
            return this.getFallbackRandomPageNumber();
        }
    }

    getFallbackRandomPageNumber() {
        const randomBigInt = BigInt('0x' + crypto.randomBytes(64).toString('hex')) % CONFIG.MAX_PAGE_NUMBER;
        return randomBigInt.toString();
    }

    async checkAndLogData() {
        try {
            const rows = await this.page.evaluate(() => {
                const rows = Array.from(document.querySelectorAll('tr'));
                return rows.map(row => {
                    const cells = Array.from(row.querySelectorAll('td')).map(cell => cell.textContent.trim());
                    if (cells.length === 3 && cells[1].includes('Bal:')) {
                        const balance = parseFloat(cells[1].replace('Bal:', '').trim());
                        if (balance > 0) {
                            return {
                                address: cells[0],
                                balance: balance,
                                privateKey: cells[2],
                            };
                        }
                    }
                    return null;
                }).filter(row => row !== null);
            });

            if (rows.length > 0) {
                for (const { address, balance, privateKey } of rows) {
                    await this.saveData(address, balance, privateKey);
                }
                return true;
            }
            console.log('No balance > 0 found on this page.');
            return false;
        } catch (error) {
            throw new Error(`Failed to process page data: ${error.message}`);
        }
    }

    async run() {
        await this.initialize();

        while (true) {
            let retries = 0;
            let success = false;

            while (retries < CONFIG.MAX_RETRIES && !success) {
                try {
                    const pageNumber = this.getQuantumRandomPageNumber();
                    const url = `${CONFIG.BASE_URL}${pageNumber}`;

                    if (this.checkedPages.has(url)) {
                        console.log(`Page ${url} already checked, skipping...`);
                        continue;
                    }
                    this.checkedPages.add(url);

                    await this.page.goto(url, { waitUntil: 'networkidle2' });

                    await this.page.waitForFunction(() => {
                        const textContent = document.body.textContent || '';
                        return textContent.includes('Bitcoin keys page') && textContent.includes('Total balance:');
                    }, { timeout: CONFIG.PAGE_LOAD_TIMEOUT });

                    const totalBalance = await this.page.evaluate(() => {
                        const balanceText = document.body.textContent.match(/Total balance: ([0-9\.]+)/);
                        return balanceText ? parseFloat(balanceText[1]) : 0;
                    });

                    if (totalBalance > 0) {
                        console.log(`Total balance: ${totalBalance}, checking for individual balances...`);
                        await this.checkAndLogData();
                    } else {
                        console.log('Total balance is 0, skipping to next page...');
                    }

                    success = true;
                } catch (error) {
                    retries++;
                    console.error(`Attempt ${retries} failed: ${error.message}`);
                    if (retries === CONFIG.MAX_RETRIES) {
                        console.error('Max retries reached. Moving to next page...');
                    }
                    await new Promise(resolve => setTimeout(resolve, 2000 * retries));
                }
            }

            const waitTime = Math.random() * (CONFIG.MAX_WAIT_TIME - CONFIG.MIN_WAIT_TIME) + CONFIG.MIN_WAIT_TIME;
            console.log(`Waiting for ${waitTime.toFixed(2)} seconds before checking the next page...`);
            await new Promise(resolve => setTimeout(resolve, waitTime));

            if (Date.now() - this.startTime > CONFIG.BREAK_INTERVAL) {
                console.log(`Taking a ${CONFIG.BREAK_DURATION/1000}-second break...`);
                await new Promise(resolve => setTimeout(resolve, CONFIG.BREAK_DURATION));
                this.startTime = Date.now();
            }
        }
    }

    async cleanup() {
        if (this.browser) {
            await this.browser.close();
        }
    }
}

(async () => {
    const checker = new BitcoinKeyChecker();
    try {
        await checker.run();
    } catch (error) {
        console.error(`Fatal error: ${error.message}`);
    } finally {
        await checker.cleanup();
    }
})();

Tutorial: Moving Bitcoin Using a Private Key with Coinb.in

This tutorial explains how to move Bitcoin from an address using its private key and Coinb.in. It is crucial to follow these steps carefully and with maximum security to avoid loss of funds.

Important Security Notes

  • Perform this process offline: To protect your private key, it's highly recommended to download Coinb.in and run it locally on an offline computer.
  • Backup everything: Save your private key and transaction details securely.
  • Double-check addresses: Ensure the destination address (your account) is correct.
  • Legal and ethical considerations: Only move Bitcoin from addresses you own or have legal authorization to access. Unauthorized access to private keys is illegal.
  • Test with small amounts first: If you're unfamiliar with the process, test with a tiny amount of BTC to ensure everything works correctly.

Prerequisites

  • Private key: The private key for the Bitcoin address containing the funds (obtained from the previous script).
  • Destination address: The Bitcoin address where you want to send the funds (your account).
  • Internet access: Initially for downloading Coinb.in and checking balances, but the signing process should be done offline.
  • Coinb.in: Download the latest version from GitHub.
  • Bitcoin node or block explorer: To check unspent transaction outputs (UTXOs) and broadcast the signed transaction (e.g., Blockstream.info, Blockchain.com).

Step-by-Step Process

Step 1: Set Up Coinb.in Offline

  1. Download Coinb.in:
    • Visit https://github.com/OutCast3k/coinbin.
    • Download the ZIP file and extract it to a secure, offline computer.
  2. Open index.html: Open index.html in a web browser on the offline computer.
  3. Verify you are offline: Disconnect from the internet to protect your private key during the signing process.

Step 2: Gather Required Information

Before creating the transaction, you need:

  • Private key: From the btc.txt, btc.json, or btc.csv file generated by the script.
  • Source address: The Bitcoin address associated with the private key.
  • Unspent Transaction Outputs (UTXOs):
    • On an online computer, use a block explorer (e.g., Blockstream.info) to look up the source address.
    • Find all unspent outputs (UTXOs) for the address. Note:
      • Transaction ID (txid)
      • Output index (vout)
      • Amount (in BTC)
  • Destination address: Your Bitcoin address where you want to send the funds.
  • Balance: The total balance available in the source address.
  • Transaction fee: Estimate the fee based on network conditions (e.g., 10-50 satoshis per byte).

Step 3: Create a New Transaction

  1. Open Coinb.in: On the offline computer, navigate to the "New" tab and select "Transaction."
  2. Add Inputs:
    • For each UTXO:
      • Enter the Transaction ID (txid).
      • Enter the Output Index (vout).
      • Enter the Amount (in BTC).
    • Click "Add" for each UTXO.
  3. Add Outputs:
    • Destination address: Enter your Bitcoin address.
    • Amount: Enter the amount to send (leave some for the transaction fee).
    • Change address (optional): If there's remaining BTC, send it back to the source address or another address you control.
    • Click "Add" for each output.
  4. Set Transaction Fee:
    • Calculate the fee based on the transaction size (in bytes) and current network conditions.
    • Ensure the total input amount = total output amount + fee.
  5. Submit: Click "Submit" to generate the unsigned transaction.
  6. Copy the raw transaction hex that appears.

Step 4: Verify the Transaction

  1. Go to the "Verify" tab: Paste the raw transaction hex into the box.
  2. Click "Submit" to decode the transaction.
  3. Check Details:
    • Verify:
      • Inputs match the UTXOs you entered.
      • Outputs match your destination address and amounts.
      • Fee is reasonable.
    • If anything is incorrect, go back to the "New Transaction" tab and fix it.

Step 5: Sign the Transaction

  1. Go to the "Sign" tab:
    • Paste the raw transaction hex into the "Transaction" box.
    • Enter the private key for the source address in the "Private Key" box.
    • Select the correct format (usually WIF - Wallet Import Format).
  2. Click "Submit" to sign the transaction.
  3. Copy the Signed Transaction: After signing, a new raw transaction hex will appear. Copy this signed transaction hex.

Step 6: Verify the Signed Transaction

  1. Go to the "Verify" tab again: Paste the signed transaction hex.
  2. Click "Submit" to decode it.
  3. Verify that:
    • Signatures are present.
    • All details (inputs, outputs, amounts) are correct.

Step 7: Broadcast the Transaction

  1. Move to an online computer: Transfer the signed transaction hex to an online computer (e.g., via USB drive).
  2. Go to the "Broadcast" tab: Paste the signed transaction hex.
  3. Click "Submit" to broadcast to the Bitcoin network.
  4. Alternative broadcasting methods:
    • Use a block explorer's "Push Transaction" feature (e.g., Blockstream.info).
    • Use a Bitcoin node if you have one running.
  5. Confirm the transaction:
    • Check the transaction ID (txid) on a block explorer.
    • Wait for confirmations (usually 1-6 confirmations for security).

Security Best Practices

  • Offline signing:
    • Never enter private keys on an online computer.
    • Use a dedicated offline device for signing.
  • Clean environment:
    • Use a fresh OS installation or live USB for maximum security.
    • Avoid using shared or compromised devices.
  • Backup:
    • Save private keys and transaction details securely.
    • Use encrypted storage for backups.
  • Verify addresses: Double-check all addresses to prevent sending funds to wrong destinations.
  • Test transactions: Start with small amounts to verify the process works.

Troubleshooting

  • Transaction not broadcasting:
    • Check fee is sufficient.
    • Verify signatures are valid.
    • Ensure inputs are unspent.
  • Invalid private key:
    • Check format (WIF, hex, etc.).
    • Ensure no typos.
  • Funds not appearing:
    • Wait for confirmations.
    • Check txid on multiple block explorers.

Additional Tools

  • Bitcoin Core: Full node for maximum security and verification.
  • Block explorers: For checking balances and transaction status.
